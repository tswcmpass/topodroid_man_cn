导入地理点

此对话框从地理点窗口的导入按钮中打开。

通过此对话框，您可以导入由支持的GNSS应用程序获取的点的坐标到TopoDroid中。

MobileTopographer
TopoDroid从MobileTopographer点列表文件中读取点坐标。
如果您没有选择MobileTopographer点列表文件，应用程序的行为是不确定的。

  1. 打开MobileTopographer并查找点坐标。
  2. 当您在坐标中达到足够的准确性时，保存该点。
  3. 您还可以手动输入坐标（并保存该点）。
  4. 保存点列表。列表保存在"MobileTopographer/pointlist"中
  5. 在TopoDroid中导入坐标。保存的点会显示在列表中；点击点以导入。

选择性地，您可以进入MobileTopographer中保存的点列表， 并使用箭头将您刚刚获取的点移到顶部。

GPS Logger
该应用程序显示点坐标和其他信息。 轨迹文件保存在"gnss_log"文件夹中。它们具有带有设备信息的头文件，以及CSV记录中的点数据： 星座、纬度、经度、高度、速度、准确性和时间。
日志还可以包含原始GNSS测量、导航消息和NMEA语句。

  1. 开始记录轨迹点。
  2. 停止轨迹记录。
  3. 完成轨迹。将轨迹添加到"轨迹列表"。

GPSTest
该应用程序显示点坐标和卫星数据：定位、星座、载波频率、载波/噪声比、标志、仰角、方位角。 可以在"设置"中选择在地图上显示该点（选项）。

  1. 打开共享/导出对话框。
  2. 可将点坐标复制到剪贴板
  3. 或保存到文件。重要的是要选中"包括高度"框。

GPX Recorder
点坐标保存在一个TopoDroid可以读取的轨迹文件中。 轨迹文件位于"gpx_recorder"文件夹中。 格式是xml，点数据为纬度、经度、高度和时间。

  1. 开始记录点坐标
  2. 停止记录。

GPS Point
TopoDroid从导出到点文件的坐标中读取：纬度、经度、高度和准确性。

  1. 该应用程序显示已保存点的列表
  2. 打开GNSS界面
  3. 获取GNSS位置
  4. 应用程序在地图上显示点的不确定圆圈。点坐标和其他信息显示在框中。
  5. 完成后保存该点。
  6. 从列表中选择该点，并导出到（文本）文件。

GPS Position
该应用程序显示点坐标和卫星数据。

  2. 显示卫星数据：定位、方位角、仰角、载波/噪声比和标志（星历、星历）
  3. 位置可以复制到剪贴板
  4. 或保存到文件。
