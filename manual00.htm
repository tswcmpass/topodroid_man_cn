TopoDroid 用户手册

marco.corvi@gmail.com


TopoDroid是一款用于洞穴测量的Android应用程序。TopoDroid将原始数据导入并帮助您组织它们，在现场生成测量线路图。您还可以在测量过程中绘制草图，从而减少绘图中可能出现的错误，并在洞穴内时检测数据错误。TopoDroid拥有许多功能，帮助您在洞穴内高效工作。因此，在测量结束时，几乎可以立即获得一张可用的地图。

TopoDroid使您在洞穴测量时更加高效。TopoDroid的目的是帮助测量员更准确地收集数据，并使测量任务更加愉快，而不是生成洞穴的最终地图。因此，已经有一些优秀的PC程序可以实现这一目的。

TopoDroid有五个不同的“活动级别”。较低的级别已经具备足够的功能，可以下载数据，管理站点名称，并绘制简单的草图。较高级别则可用于准备数据和草图，以便在PC上更轻松地进行后续处理。

TopoDroid设计用于一次处理一个测量项目，而不是管理包含多个不同测量的复杂洞穴测量项目。尽管如此，它包含一个“洞穴项目管理器”，可用于将测量组合成一个洞穴/喀斯特项目，以及一个“3D查看器”，可用于查看测量和洞穴项目的3D效果。

TopoDroid支持DistoX（v. 1、2和BLE）、Sap5和Bric4。然而，测量数据也可以手动输入。
请注意，如果Android设备（以及主动触控笔）离电子设备太近，它们可能会影响电子设备的测量。

TopoDroid 私人文件夹

TopoDroid的私人文件夹位于 "Android/data/com.topodroid.TDX/files"， 其中包含绘图工具的文件夹 "point"、"line" 和 "area"，以及设备数据库 "device10.sqlite"，其中包含校准数据， 导出的校准文件（文件夹 "ccsv"），DistoX2和DistoXBLE固件文件（文件夹 "bin"）， 以及DistoX的内存转储（文件夹 "dump"）。

此文件夹及其内容在卸载应用程序时将被删除，除非您选择保留应用程序数据的选项。

TopoDroid 测量文件夹

TopoDroid的基础目录是 "Documents/TDX"，位于<主要外部存储>中。 它包含项目文件夹 "TopoDroid"。项目文件夹包含数据库 "distox14.sqlite"，"zip" 文件夹，用于临时文件的文件夹 "tmp"， 并且针对每个创建的测量，该文件夹都有一个相应的子文件夹。

在卸载应用程序时，此文件夹及其子文件夹不会被删除。


测量数据

测量数据和草图保存在 TDX 的 "TopoDroid" 子文件夹中。
每个测量在 "TopoDroid" 下有一个子文件夹。
"TopoDroid" 文件夹包含数据库 "distox14.sqlite"，测量文件夹， 以及洞穴项目文件夹 "thconfig"。

授予权限

在Android-11及更高版本中，TopoDroid可能无法创建基础文件夹 "TDX"，或其中的一些子文件夹或文件， 因为它没有足够的权限来访问<主要外部存储>。
在这种情况下，您可以使用 "adb"（Android调试桥）授予权限。
  1. 在您的PC上下载 "adb"
  2. 在Android设备上，启用 "开发者选项"（进入设置 "关于手机"，连续点击 "版本号" 七次）
  3. 在 "开发者选项" 中启用 "USB调试"
  4. 在PC上启动 adb 服务器。输入命令 "adb start-server"
  5. 使用USB线将Android设备连接到PC。您可能需要在Android上允许来自PC的adb连接
  6. 在PC上输入三个命令：
    • adb shell appops set com.topodroid.TDX READ_EXTERNAL_STORAGE allow
    • adb shell appops set com.topodroid.TDX WRITE_EXTERNAL_STORAGE allow
    • adb shell appops set com.topodroid.TDX MANAGE_EXTERNAL_STORAGE allow

导入旧测量数据

TopoDroidX（v.6）可以与旧版本v.5.1.40同时安装，因为它们使用不同的基础文件夹。
可以从旧TopoDroid创建的zip存档中导入旧v.5.1.40测量数据到v.6。
还可以使用位于以下网址的zip脚本创建这些zip存档：
http://marcocorvi.altervista.org/caving/speleoapps/speleoapks/TopoDroidZipScript.html

N.B. 由于数据库更改，TopoDroid v.5.1.40无法导入v.6的zip存档。

硬件特性

TopoDroid使用以下硬件特性："蓝牙"、"相机"、"麦克风"、"GNSS"和"多点触控"。 如果TopoDroid需要您的Android设备没有的某个特性，相关功能将自动关闭。
Internet连接仅用于安装用户手册的翻译以及检查是否有新版本可用。
Android权限

本地库

TopoDroid包含为选定架构（arm64_v8a、armeabi-v7a和x86_64）编译的一些本地库。 这些库包括：powercrust 和 tiff，其中的 libtiff 包含了 libjpeg 和 libpng 的静态链接。
