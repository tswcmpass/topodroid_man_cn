PHOTO-ITEM EDIT

This dialog displays the properties of sketch points of type "photo".

  • thumbnail of the photo
  • orientation (azimuth and clino) of the photo, for photos taken with TopoDroid
  • date and time of the photo
  • comment (editable)

Buttons:

  • Save saves the changes to the comment

The image is displayed when you tap on the thumbnail. The full image is not reoriented to portrait.

The Android BACK button closes the dialog.
