3D SKETCHING

3D sketching is done around one leg at a time. Therefore the display is centered about the selected leg.

The 3D Sketch window has three modes (and three different button bars):

  • move
  • draw
  • select (only for the "leg" view)
As for the 2D Drawing window, the leftmost green buttons indicate the current mode.
The other buttons for the move mode are:
  • display mode
  • notes
  • walls
  • refresh
In the draw mode you can trace contour lines. The contour line of the "leg" view lie in the 3D plane dipping with the leg. Those of section views lie in the section plane.

Buttons:

  • undo
  • redo
The erase mode has buttons
  • undo
  • redo
  • eraser size

For the select mode:

  • selection mode: either contour/section points or neighbor stations
  • select size
  • delete
  • open: either section or neighbor "leg" view
  • clear selection

The "leg" view contains also the connected legs. Tapping one of neighbor stations moves the "leg" view to the leg of that station.

A section can be created when two points on two contour lines have been selected. The section plane can be "vertical" or "horizntal". If it is vertical it contains the two selected points, and the Y-axis is the vertical. If it is horizontal it passes through their midpoint and the Y-axis is the leg azimuth. The two points (or their projections) are shown in the display.
A menu commutes between the "vertical" and the "horizontal" choices.

When the canvas displays a x-section the "select" button turns into "back" button. A tap closes the x-section and goes back to the leg view.

A x-section can be reopened by selecting the point of the x-section (midpoint between the two x-section base points) ant tappint the "open" button. Menus

  • Close
  • Export (ie, save)
  • Zoom fit
  • Section type
  • Delete
  • Settings
  • Help
