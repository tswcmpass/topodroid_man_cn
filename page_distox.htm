DistoX 参考指南

以下是 DistoX 功能的摘要。
官方用户手册可在 http://paperless.bheeb.ch 获取。

PIN 码：
如果 DistoX 未自动配对，请使用 PIN 码 "0000"（四个零）。

错误代码：

  • 252：温度过高
  • 253：温度过低
  • 255：信号太弱
  • 256：信号太强
  • 257：背景光线太强
  • 260：激光束中断

主要功能：

  • DIST：开机 / 启动激光 / 测量
  • CLR：取消当前操作，关闭激光
  • CLR（长按2秒）：关闭设备
  • MEM：内存条目
  • REF：切换距离参考

DistoX2及DistoXBLE

  • TIMER（激光开启）：启动计时器（自动测量）
  • TIMER（激光关闭）：设置计时器间隔（使用 PLUS/MINUS）
  • SMART：额外测量信息
  • FUNC：设备信息

两秒操作：

  • DIST（激光开启）：测量
  • CLR：关机
  • CLR+SMART：切换校准模式
  • CLR+MEM：清除未发送的内存
  • CLR+FUNC：切换蓝牙
  • CLR+MINUS：锁定关机
  • REF：前方参考
  • REF+FUNC：后方参考
  • REF+PLUS：切换显示灯
  • REF+MINUS：切换蜂鸣
  • MEM：距离单位
  • MEM+SMART：角度单位（度/梯度）
  • MEM+FUNC：切换静音模式（测量不传输）
  • MEM+MINUS：切换反测视模式 [2.4]
  • SMART+MINUS：切换三次测量检查 [2.4]
  • DIST+PLUS：连续测量，CLEAR 关闭 [2.6]

五秒操作：

  • FUNC+SMART：更改电池类型
  • FUNC+CLR+MEM：恢复出厂设置
  • DIST+PLUS+MINUS（开机时同时按下）：引导模式

额外测量信息：

  1. 方位角、水平距离、垂直距离、距离
  2. 横滚角、俯仰角
  3. M、G
  4. 内置加速度计原始值 x、y、z (*)
  5. 独立加速度计原始值 x、y、z (*)
  6. 磁场原始值 x、y、z (*)

  1. 组合加速度计 x、y、z
  2. 磁场 x、y、z
  3. 内置加速度计 x、y、z (*)
  4. 独立加速度计 x、y、z (*)

设备信息：

  1. 电池
  2. 硬件/固件版本、序列号
  3. 显示灯级别（1:10）[2.3]
  4. 蜂鸣音调（0:5）[2.6]
  5. 端点偏移量（-126:127）[2.3]

DistoX - A3

  • AREA（背向参考）：持续显示方位角+俯仰角 (*)
  • AREA+REF（前向参考）：持续显示横滚角+俯仰角 (*)
  • AREA*2：固件版本和序列号
  • UNITS：切换显示灯
  • AREA*2+REF*2+CLR：切换静音模式
(*) 请勿在此模式下进行测量

两秒操作：

  • UNITS（在背向参考中）：距离单位
  • UNITS（在前向参考中）：循环角度单位 / 蓝牙
  • MEM*10+UNITS：切换校准模式

五秒操作：
  • CLR+PLUS：切换蜂鸣

"角度单位 / 蓝牙" 循环
  • 关闭指南针，关闭蓝牙
  • 开启指南针（度），关闭蓝牙
  • 开启指南针（梯度），关闭蓝牙
  • 开启指南针（度），开启蓝牙
  • 开启指南针（梯度），开启蓝牙
