3D VIEWER - Fractal Analysis

The fractal analysis considers the 3D space divided into cubes and counts the number of cubes that cover the cave (midline and splays). For a 3-dimensional object, when the cube side halves, i.e. is divided by 2, the number of cubes that cover the cave increase by 8 = 23. The fractal exponent at a given side is the power of 2 (ie, 3 in the example).

The fractal analysis finds this exponent for varying length of the cube side, and displays the result as a graph.

The Fractal dialog is opened from the Fractal menu of the 3D Viewer window.
It displays the result of the last fractal computation or starts a new fractal computation.

The result is a graph plotting the variation of the computed dimension as the size of the unit cubic cell varies.

The computation depends on

  • counting mode
  • whether to consider also the splays or only the legs
  • the starting cell size

Counting
There are three counting modes:
  • counting the number of filled boxes
  • counting the number of filled box-pair along X, Y, or Z directions (6 pairs)
  • counting the number of filled box-pair along X, Y, or Z as well as the diagonal directions (26 pairs)

The cell (box) size is increased by sqrt(2) at each step.
